# Key Decisions
- **reducer 是投影唯一写入者**：handler（Step 6b+）只调 writeEvent + applyAssessmentEvent，不另 INSERT

- **幂等策略**：INSERT 类用 PK 存在性判；UPDATE 类用 last_status_event_id == event.event_id（单调重放下充分）
